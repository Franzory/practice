## Build Setup

``` bash

// install dependencies
npm install

// serve with hot reload at localhost:8080
npm run dev

// build for production with minification
npm run build

```


## example:
```
<template>
    <div>
        <setting></setting>
    </div>
</templage>

<script>
  ...
  import setting from "vue-dynatic-icon"
  ...
  
  export default{
      data() {},
      components:{
        setting
      }
  }
  
</script>
```
